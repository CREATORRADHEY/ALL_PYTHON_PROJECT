# All Python Projects

This repository contains a collection of mini-projects, scripts, and applications written in Python. It serves as a portfolio of various functionalities and implementations.

## Projects & Scripts

### Applications

- **`JARVIS/`**: An AI Voice Assistant capable of performing various tasks.
- **`Game/`**: A Tic-Tac-Toe game implementation.
- **`file_transfer_app.py`**: A GUI application for transferring files between directories.
- **`POSITIVE_NEGATIVE_CHECKER/`**: A simple utility to check if a number is positive or negative.

### Communication

- **`EMAIL.PY`**: A script to send emails using SMTP.
- **`sms.py`**: A script to send SMS messages (likely using Twilio or similar service).
- **`call.py`**: A script to initiate phone calls.

### Utilities & Learning

- **`web_search.py`**: A utility script for performing web searches.
- **`speaker.py`**: A text-to-speech utility.
- **`Introduction/`**: Contains introductory Python scripts and examples.
- **`LIST&TUPLE/`**: Examples and exercises demonstrating Python lists and tuples.

### Other Files

- **`PyWhatKit_DB.txt`**: Database file for PyWhatKit library.
- **`static/`**: Directory containing static assets (CSS, images).

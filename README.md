## Flask Portfolio Details

A personal portfolio website built with Flask.

### Description

This project is a simple web application designed to showcase a portfolio, resume, and about section. It uses Flask as the backend framework to serve HTML templates.

### Features

- **Home Page**: Landing page of the portfolio.
- **About Page**: Information about the creator.
- **Portfolio Page**: Showcase of projects and work.
- **Resume Page**: Professional resume display.

### Tech Stack

- **Backend**: Python (Flask)
- **Frontend**: HTML, CSS

### Installation (Standalone)

1. Clone the repository:

   ```bash
   git clone https://github.com/CREATORRADHEY/FLASK_PORTFOLIO.git
   ```

2. Navigate to the project directory:

   ```bash
   cd FLASK_PORTFOLIO
   ```

3. Install dependencies (if any, e.g., Flask):
   ```bash
   pip install flask
   ```

### Usage

Run the application:

```bash
python FLASK.PY
```

The application will start on `http://127.0.0.1:5003/`.
